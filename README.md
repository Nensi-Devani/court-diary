# Court-Diary

This is a full-stack **MERN** (MongoDB, Express, React, Node.js) application.

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Nensi-Devani/court-diary
cd court-diary
cd client
```
### 2. Install node_modules
```bash
npm i
```
### 3. Run the project
```bash
npm run dev
```
