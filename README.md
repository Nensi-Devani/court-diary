# Court-Diary

This is a full-stack **MERN** (MongoDB, Express, React, Node.js) application.

## Getting Started

### 1. Pull the code & Open 2 different terminals
```bash
git pull
```
### 2. Install required packages
```bash
npm i
```
### 3. Terminal 1 to run frontend
```bash
cd client
npm run dev
```
### 4. Terminal 2 to run backend
```bash
cd server
npm start
```
